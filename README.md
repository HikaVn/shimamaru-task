# shimamaru-task 🐦

しままる（シマエナガ）と いっしょに タスクを すすめる、**ADHDフレンドリーな育成タスクアプリ**。
UIには明記しないものの、短期報酬・進捗の可視化・マイクロタスク・時間盲対策・ボディダブリング・
「絶対に罰しない」設計で、毎日のタスク遂行を後押しします。

## アプリ本体
- `simamaru_memo.html` … 単一ファイルのWebアプリ（依存なし・localStorage保存・PWA対応）
  - おうち（しままる育成）／おしごと（タスク・ひなタスク）／カレンダー／ずかん（ひな図鑑・バッジ）／きろく（進捗）
  - 育成/RPG：XP・レベル・木の実・たまご孵化・ショップ・れんぞく日数
  - しゅうちゅうタイム（視覚タイマー＋ボディダブリング）、デイリーミッション、くりかえしタスク（ルーティン）
  - 外部アプリ連携（しままるタイピング / コドリ）、まいにちリマインド通知
- iPhoneは Safari →「ホーム画面に追加」で アプリ風に使えます。

## ツール（`tools/`）
- `make-icons.js` … 正方形PNGから PWA各サイズ（192/512/180/32）を生成（依存なし）
- `gen-icon-openai.js` … OpenAI画像API(gpt-image-1)でアイコンを生成（要 APIキー＆egress許可）

## MCPサーバ（`mcp-server/`）
外部AI（Claude Desktop / Claude Code / Codex）から、しままるのタスクを操作できる MCP サーバ。
詳細は [`mcp-server/README.md`](mcp-server/README.md) を参照。
