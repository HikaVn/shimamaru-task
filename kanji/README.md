# しままる漢字 📖

中1〜高3の漢字を **読めるようになる** ための、しままると一緒に進む冒険RPG。
単一ファイル（`kanji/index.html`）・依存なし・localStorage保存。

- 公開URL（GitHub Pages）: `https://hikavn.github.io/shimamaru-task/kanji/`
- タスクアプリの「🎮 あそび・れんしゅう」からもリンクしています。

## あそびかた
- 🗺️ **ぼうけん**：6つのエリア（みどりの森＝中1相当 … ほしぞらの塔＝高3相当）を進む。
  漢字モンスターが出す**熟語の読み**を4択で当てる。正解で なかまに、まちがえても
  やさしく正解を見せて **また出会える**（ぜったいに罰しない）。各エリアの最後はボス。
- 📚 **ずかん**：エリアごとに 漢字・読み・意味を一覧。「よみをかくす」で自分テストも。
- 🔁 **ふくしゅう**：まちがえた・あやしい漢字を集めて出題（軽いSRS）。

## 育成
- 正解でXP → しままるレベルアップ。`📖 読めた数`（=2回正解で習得）と `🤝 なかま` を可視化。

## データについて
- 読みは**熟語で出題**します。**学年の区分は目安**です（中学・高校で習う常用漢字の難易度順）。
- 出題語はMVPの厳選セット（各エリア12語＝計72語）。`index.html` 内の `STAGES` 配列に
  追記するだけで簡単に増やせます。

## 注意
- まちがいは「失敗」ではなく「もう一度であえるチャンス」。HP減少や罰はありません（しままる流）。
