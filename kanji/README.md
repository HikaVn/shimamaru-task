# しままる漢字 📖

中1〜高3の漢字を **読めるようになる** ための、しままると一緒に進む冒険RPG。
単一ファイル（`kanji/index.html`）・依存なし・localStorage保存。

- 公開URL（GitHub Pages）: `https://hikavn.github.io/shimamaru-task/kanji/`
- タスクアプリの「🎮 あそび・れんしゅう」からもリンク。各画面の 🏠 から しままるタスクへ戻れます。

## こうぞう（2階層）
**ワールド（学年）→ ステージ（小面）** の2階層で、少しずつクリアして進みます。
- **6ワールド**（みどりの森＝中1相当 … ほしぞらの塔＝高3相当）
- 各ワールド **3ステージ**、各ステージ **10語** ＝ **計180語**
- ワールドは前のワールドを全クリアで解放、ステージは順番に解放

## あそびかた
- 🗺️ **ぼうけん**：ワールドをタップ→ステージ選択→4択で熟語の読みを当てる。
  正解で なかまに、まちがえても やさしく正解＆**再出題**（ぜったいに罰しない）。
  各ステージにボス＋**隠しおたから🎁**。
- 💎 **おたから**：各ステージのたからばこを集めるコレクション（全18種）。
  **パーフェクト**（全問正解）で ✨キラ✨ 版に！
- 📚 **ずかん**：ワールド別に 漢字・読み・意味を一覧（「よみをかくす」で自己テスト）。
- 🔁 **ふくしゅう**：まちがえた・あやしい漢字を集めて出題（軽いSRS）。
- 育成：正解でXP→Lv、`📖 読めた数`（2回正解で習得）と `💎 おたから` を可視化。

## データについて
- 読みは**熟語で出題**。**学年の区分は目安**（常用漢字の難易度順）。
- 現在 **180語**（各ワールド30語）。`index.html` 内の `WORLDS` 配列の各ステージ `items` に
  追記するだけで増やせます（同じワールド内で読みが重複しないようにだけ注意）。
- ※ データ構造刷新のため保存キーは `shimamaru-kanji-v2`（旧v1の進捗は引き継ぎません）。

## 注意
- まちがいは「失敗」ではなく「もう一度であえるチャンス」。HP減少や罰はありません（しままる流）。
